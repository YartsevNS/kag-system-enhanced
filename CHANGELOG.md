# 📝 KAG CHANGELOG

## Версия 0.3.0 (2026-04-12)

### 🔐 Безопасность

#### ✅ SSH подключения с шифрованием

**Добавлено:**
- Настройки SSH подключения хранятся в зашифрованном виде
- Используется GOST шифрование для защиты паролей
- Файл настроек имеет права доступа `600` (только для владельца)
- Пароли не передаются обратно в API (только индикатор наличия)

**Новые endpoints:**
- `GET /api/v1/admin/models/ssh-config` - Получить настройки (без паролей)
- `POST /api/v1/admin/models/ssh-config` - Сохранить настройки (пароли шифруются)
- `POST /api/v1/admin/models/ssh-test` - Протестировать SSH подключение

**Файлы:**
- `src/api/services/ssh_manager.py` - Менеджер SSH подключений
- Шифрование: `src/security/gost_crypto.py`
- Хранилище: `/app/data/ssh_connections.json` (зашифровано)

#### 🔑 Ключ шифрования

- Генерируется автоматически при первом запуске
- Сохраняется в `/app/data/.encryption_key`
- Права доступа: `600` (только для владельца)
- Можно задать через переменную окружения `SSH_ENCRYPTION_KEY`

### 🎨 Веб-интерфейс

#### ✅ Форма настрое SSH

**Добавлена секция "Настройки SSH подключения" на странице админки:**
- IP адрес сервера Ollama
- SSH порт (по умолчанию 22)
- Имя пользователя
- SSH пароль (шифруется при сохранении)
- Пароль для sudo (опционально)
- Порт Ollama API (по умолчанию 11434)
- Имя сервиса Ollama

**Кнопки:**
- 💾 Сохранить настройки
- 🔍 Тест подключения
