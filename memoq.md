## 🐛 Исправленные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `❌ Ошибка сохранения` | Middleware Setup редиректил запросы | Добавлены исключения для `/api/v1/admin/models/*` |
| `mistral:7b` в шапке | Хардкод в `index.html` | Заменено на динамическую загрузку из API |
| `sshpass: not found` | Отсутствовал в Dockerfile | Добавлен `openssh-client` и `sshpass` |
| `time: missing unit in duration "-1"` | Ollama требовал единицы времени | Заменено на `"24h"` |
| `await outside async function` | Синтаксическая ошибка в model_manager.py | Добавлен `async` к методам инициализации |
| Kong Error / Auth required | Auth middleware блокировал Setup | Настроены публичные пути |
