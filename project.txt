ОБЗОР ПРОЕКТА

Проект kag — полная замена классического RAG. Обеспечивает многоагентную обработку знаний, пакетную индексацию мультимодальных документов (PDF, таблицы, аудио), централизованный API-шлюз, веб-интерфейсы, модуль оценки качества и соответствие 152-ФЗ.
Стек: Python 3.11, FastAPI, Qdrant, Redis, Celery+APScheduler, Keycloak(OIDC/LDAP)+Casbin(RBAC), Nginx, Ollama/vLLM, OpenTelemetry, MCP 1.2.0.
Поток данных: Пользователь -> Nginx(auth_request->Keycloak) -> FastAPI(Casbin RBAC) -> Qdrant/Redis -> Celery Worker -> Ollama/LLM -> Ответ. Логи -> Loki -> SIEM. Метрики -> Prometheus -> Grafana.

Криптография: безопасная генерация/загрузка ключа, шифрование AES-256-GCM (fallback ГОСТ).
import os, logging
from pathlib import Path
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
logger = logging.getLogger(name)
class CryptoManager:
    def init(self, key_path: str = "/app/data/gost.key"):
        self.key_path = Path(key_path)
        self._key = None
        self._aesgcm = None

def get_or_create_key(self) -> bytes:
    if self._key: return self._key
    if self.key_path.exists():
        self._key = self.key_path.read_bytes()
        return self._key
    logger.info("Генерация нового ключа шифрования...")
    key = AESGCM.generate_key(bit_length=256)
    self.key_path.parent.mkdir(parents=True, exist_ok=True)
    self.key_path.write_bytes(key)
    self.key_path.chmod(0o600)
    self._key = key
    return key

def encrypt(self,  bytes) -> dict:
    nonce = os.urandom(12)
    ct = AESGCM(self.get_or_create_key()).encrypt(nonce, data, None)
    return {"nonce": nonce.hex(), "ciphertext": ct.hex(), "algo": "AES-256-GCM"}

def decrypt(self, payload: dict) -> bytes:
    return AESGCM(self.get_or_create_key()).decrypt(bytes.fromhex(payload["nonce"]), bytes.fromhex(payload["ciphertext"]), None)

crypto = CryptoManager(os.getenv("GOST_KEY_PATH", "/app/data/gost.key"))

@mcp.tool()
async def get_system_status() -> dict:
    """Диагностика: Qdrant, Redis, Ollama, версии."""

        async with httpx.AsyncClient(timeout=5) as c:
