🔗 Гибридный поиск (граф + вектор)

Ищет одновременно в двух пространствах: Neo4j (граф — сущности и их связи) и Qdrant (векторы — смысловая близость чанков). Результат — чанки документов, где встречаются указанные сущности или похожий по смыслу текст. Примеры: Иванов, ООО Ромашка найдёт все документы где есть эти организация и человек; криптография, ГОСТ — документы по теме криптозащиты.
