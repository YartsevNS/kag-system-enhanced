Распознавание сканов PDF и изображений. В 20× быстрее EasyOCR, 93.7% точность. 0.57 сек/страницу.

Документы и версионирование
Загрузка PDF, DOCX, TXT. SHA-256 трекинг изменений. Авто-переиндексация при обновлении документа.

Мониторинг веб-страниц и папок
Отслеживание изменений на URL. Watchdog для локальных папок. Уведомления администратора.
Новое

Multi-user RBAC
Регистрация пользователей, группы доступа. JWT-аутентификация. Ограничение поиска по группам.
