# KAG Project - Отчёт об изменениях: OCR для русского языка (13 апреля 2026)

## ✅ ДОБАВЛЕН OCR (Tesseract)

### Выбор: Tesseract OCR (https://github.com/tesseract-ocr/tesseract)
- Open-source, проект Google
- Отличная поддержка русского языка (пакет `tesseract-ocr-rus`)
- Де-факто стандарт в индустрии

### Новые файлы:
1. `src/indexing/ocr_engine.py` — модуль OCR-движка
   - Класс `OCREngine` с методами:
     - `extract_text_from_image()` — OCR изображений
     - `extract_text_from_pdf()` — OCR PDF-документов
   - Языки: `rus+eng` (русский + английский)
   - DPI: 300 для качественного распознавания

### Изменённые файлы:
2. `src/indexing/parsers.py` — интегрирован OCR в PDF парсер
   - Если PyPDF2 не нашёл текст или он < 50 символов → автоматически OCR через Tesseract
   - Метаданные содержат флаг `ocr_used: true/false`
   - Логирование каждой страницы с OCR

3. `requirements.txt` — добавлены Python-зависимости:
   - `pytesseract>=0.3.10` — обёртка для Tesseract
   - `Pillow>=10.0.0` — обработка изображений
   - `pdf2image>=1.17.0` — рендеринг PDF страниц в изображения

4. `Dockerfile` — добавлены системные зависимости:
   - `tesseract-ocr` — основной движок
   - `tesseract-ocr-rus` — русский язык
   - `poppler-utils` — для pdf2image

### Как работает:
1. При загрузке PDF → PyPDF2 пытается извлечь текст
2. Если текст пустой или слишком короткий (< 50 символов) → Tesseract OCR
3. Каждая страница рендерится в изображение (300 DPI)
4. Tesseract распознаёт русский + английский текст
5. Результат сохраняется в сегменты с метаданными `{"ocr_used": true}`
